# Swatch — Product Overview

## Problem

Interior designers and homeowners see colors they love — in a photo, a room, a fabric — and have no easy way to find the matching paint. Current tools are brand-locked (ColorSnap = Sherwin-Williams only) or don't touch paint at all (Coolors). Getting from "I love that green" to "buy Benjamin Moore HC-114" is a multi-step, manual process.

## Target User

Interior designers, home renovators, and color-obsessed homeowners who want to go from inspiration image to paintable color in seconds.

## Core Value Proposition

Upload any image, get a color palette matched across the biggest paint brands — Sherwin-Williams, Benjamin Moore, Behr, PPG, and Valspar — plus Pantone codes. One upload, every brand, instant results.

## V0.1 Feature Set

### In

- Image upload (drag & drop or file picker)
- Automatic dominant color extraction (5-8 colors per palette)
- Match each color to nearest equivalent across 5 paint brands (Sherwin-Williams, Benjamin Moore, Behr, PPG, Valspar)
- Match each color to nearest Pantone code
- Save palettes to account (auth, up to 10 users)
- Desktop-first responsive layout
- On-screen palette display with color names, codes, and swatches

### Out (for now)

- Mobile-optimized experience
- Exports (PDF, image download)
- Room visualization / AR
- Payments / subscription
- Team sharing / collaboration
- Paint purchase links

## Tech Stack

Next.js + Supabase + Vercel + TypeScript (default stack)

## Success Metric

A user can upload a photo and get a useful, accurate paint-matched palette in under 10 seconds.

## Design Inspiration

@jennaisfeeling — bold, saturated colors, joyful and whimsical but sophisticated. Editorial quality. Retro/Wes Anderson warmth. Not minimalist — colorful with intention. The app should feel like a celebration of color, not a sterile utility.
